# 🛒 E-Commerce Frontend Website

## 📌 Project Overview

This project is a desktop-based **E-Commerce Frontend Website** developed as part of an internship task. The goal of this project is to implement a visually accurate and user-friendly interface based on a provided Figma design using core frontend technologies.

## 🚀 Features

* 🧭 **Responsive Header & Footer**
  * Navigation bar with logo, menu, and search bar
  * Structured footer with useful links

* 🏠 **Home Page**
  * Hero section with banner and call-to-action
  * Featured categories and promotional sections

* 🛍️ **Product Listing Page**
  * Grid-based product cards
  * Product image, name, price, and "Buy Now" button
  * Hover effects for better user experience

* 📦 **Product Details Page**
  * Detailed product information
  * Price display and "Add to Cart" button
  * Related products / customer review section

* ⚙️ **Basic Interactivity**
  * Dropdown for product selection
  * Styled search bar (UI-based)
  * Hover animations and effects


## 🛠️ Technologies Used
* **HTML5** – Structure of the website
* **CSS3** – Styling, layout, and animations
* **JavaScript** – Basic interactivity
